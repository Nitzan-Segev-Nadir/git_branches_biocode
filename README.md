עבודה מקבילית עם Git ו-Branches


מה נעשה בתרגיל?

נבצע סימולציה של שני תלמידים העובדים במקביל על אותו פרויקט ביואינפורמט
הפרויקט: קריאת רצף מקובץ FASTA וחישוב מאפיינים של הרצף.


שלבי העבודה:

אחד מבין ההזוג שעובד יחדיו יעשה Fork ל-repo  הבא: https://github.com/VeredKunik/git_branches_biocode


כעת נעשה Clone של ה-repository

יצירת שני branches לפיתוח במקביל

כל branch מוסיף פונקציה אחרת לקוד
ענף אחד מחשב %GC, והענף השני מחשב %AT

ביצוע Commit לשמירת השינויים

 נעבור ל- Publish/Push ל-GitHub
 
ניצור Pull Request
 נמזג (merge) את השינויים לענף הראשי

מה נלמד מהתרגיל?

איך עובדים במקביל על אותו קוד

למה משתמשים ב-branches

מהו Pull Request

איך Git  פותר או מזהה merge conflicts
